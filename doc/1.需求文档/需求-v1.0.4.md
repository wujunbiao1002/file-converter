# v1.0.4

# 需求
修复 md 转 word 时的以下 bug：
1. 转换后生成重复文字的问题。
2. 转换后列表项出现多余的圆点或序号（双重前缀）的问题。
3. 转换后列表项内部文本自动产生多余换行符的问题。

# 说明
1. 修复 HTML 文本解析时的无限递归遍历，避免段落及其内部元素的文本重复输出。
2. 修复内联格式识别逻辑，在解析 Markdown 粗体、斜体、代码、链接等格式时能正确渲染到 Word 并避免丢失。
3. 修改列表项处理逻辑，去掉人工手动添加的 `•` 和序号前缀，利用 Word 原生的 List Bullet 和 List Number 样式实现格式对齐。
4. 优化列表内嵌段落等复合文本节点的换行与空白字符处理，过滤节点间的换行符。
